Оцени новость для Telegram-канала о Fortnite от 0 до 100.

Канал публикует только интересный контент:
- скины;
- будущие скины;
- сезоны;
- боевой пропуск;
- коллаборации;
- магазин предметов;
- ивенты;
- бесплатные награды;
- утечки;
- V-Bucks.

Неинтересный технический контент нужно оценивать низко:
- проблемы серверов;
- вход;
- матчмейкинг;
- технические работы;
- восстановление сервисов.

Верни строго JSON в формате:
{
  "score": 0,
  "category": "",
  "reason": "",
  "publish": true,
  "is_leak": false,
  "is_official": false
}
