# Fortnite AI Telegram Assistant — версия под вирусные новости и рост аудитории

**Цель проекта:** создать полностью автоматизированного ИИ-ассистента для Telegram-канала, который собирает самые интересные новости по Fortnite, переписывает их на русском языке в официальном новостном стиле, генерирует привлекательные изображения и публикует посты со ссылкой на магазин V-Bucks.

**Главный фокус:** контент, который интересен зрителю и может набирать аудиторию:
- будущие скины;
- новые коллаборации;
- следующий сезон;
- боевой пропуск;
- обновления магазина предметов;
- редкие возвращения скинов;
- бесплатные награды;
- ивенты;
- утечки по косметике, картам, режимам и датам;
- трейлеры, тизеры, анонсы Epic Games;
- крупные изменения в Battle Royale, LEGO Fortnite, Fortnite Festival, OG, Reload.

**Что НЕ отслеживаем:**
- технические работы;
- проблемы с входом;
- проблемы матчмейкинга;
- задержки с V-Bucks;
- проблемы магазина;
- восстановление серверов;
- любые скучные технические статусы, которые не дают просмотры.

---

## 1. Концепция бота

Бот работает как редакция Telegram-канала:

1. Следит за источниками новостей и утечек.
2. Находит свежую информацию.
3. Оценивает, интересно ли это аудитории.
4. Проверяет достоверность через несколько источников.
5. Переводит и переписывает новость на русском.
6. Делает пост в официальном новостном стиле.
7. Генерирует изображение под пост.
8. Добавляет аккуратную ссылку на магазин V-Bucks.
9. Публикует пост в Telegram-канал.

Формат канала: **официальный новостной стиль**, без клоунады, без слишком разговорной подачи, но с понятными и цепляющими заголовками.

## 2. Типы контента для канала

### 2.1. Самый приоритетный контент

Это контент, который чаще всего собирает просмотры, репосты и обсуждения:

| Тип контента | Что публикуем | Приоритет |
|---|---|---|
| Будущие скины | новые скины, стили, наборы, коллаборации | Очень высокий |
| Следующий сезон | тема сезона, дата, карта, боевой пропуск | Очень высокий |
| Боевой пропуск | скины, награды, уровни, цена, секретный скин | Очень высокий |
| Коллаборации | Marvel, Star Wars, anime, celebrities, games, movies | Очень высокий |
| Магазин предметов | новые предметы, редкие возвращения, выгодные наборы | Высокий |
| Бесплатные награды | квесты, Twitch Drops, турниры, подарки | Высокий |
| Ивенты | live-event, мини-ивенты, финал сезона | Высокий |
| Карта и локации | новые POI, изменения карты, возвращение старых мест | Средний/высокий |
| Оружие и механики | новое оружие, транспорт, способности | Средний |
| Fortnite Crew | новый скин Crew, награды, V-Bucks | Средний/высокий |

### 2.2. Что публиковать осторожно

Утечки можно использовать, но нельзя выдавать их за подтвержденные новости.

Правильная маркировка:
- `Утечка: ...`
- `По данным датамайнеров: ...`
- `Информация пока не подтверждена Epic Games`
- `Официального анонса пока нет`

Неправильно:
- `Epic подтвердила`, если Epic ничего не подтверждала.
- `100% выйдет`, если это только слух.
- `официально`, если источник — датамайнер или Reddit.

## 4. Оценка интересности новости

Каждой найденной новости бот ставит score от 0 до 100.

### 4.1. Формула оценки

```text
score = relevance + freshness + source_trust + audience_interest + monetization_fit
```

Где:

| Параметр | Максимум | Описание |
|---|---:|---|
| relevance | 25 | Насколько тема связана с Fortnite, скинами, сезоном, V-Bucks |
| freshness | 20 | Насколько новость свежая |
| source_trust | 20 | Уровень доверия источника |
| audience_interest | 25 | Потенциал просмотров и обсуждений |
| monetization_fit | 10 | Насколько уместно добавить ссылку на магазин |

### 4.2. Приоритеты

| Score | Действие |
|---:|---|
| 85–100 | Публиковать сразу |
| 70–84 | Публиковать, если нет поста на похожую тему |
| 50–69 | Отложить в очередь или объединить в дайджест |
| 0–49 | Не публиковать |

### 4.3. Автоматический фильтр скучного контента

Бот должен пропускать новости, где есть такие темы:

```text
server status
downtime
login issue
matchmaking issue
V-Bucks delay
payment issue
item shop issue
maintenance
service outage
service restored
```

На русском:
```text
технические работы
проблемы с входом
матчмейкинг
задержка начисления
сбой магазина
серверы восстановлены
```

Исключение: если техническая тема напрямую связана с большим обновлением, новым сезоном или ивентом. Но даже тогда пост должен быть не о проблемах, а о том, что появится после обновления.

## 5. Логика публикаций

### 5.1. Автопубликация

Полностью автоматически можно публиковать:

- официальные новости Epic/Fortnite;
- новый трейлер с официального YouTube;
- новый боевой пропуск;
- обновление магазина предметов;
- появление новых косметических предметов в API;
- бесплатные награды;
- официально подтвержденные коллаборации.

### 5.2. Утечки

Утечки публикуются автоматически только если выполнено одно из условий:

1. Утечку подтвердили минимум 2 надежных источника.
2. Предмет найден в базе Fortnite-API/Fortnite.GG.
3. Изображения и ID предметов появились в игровых файлах.
4. Об этом написали HYPEX/ShiinaBR/FireMonkey, и тема имеет высокий интерес.

В каждом таком посте обязательно:
- указать, что это утечка;
- не писать “официально”;
- добавить фразу `Epic Games пока не подтверждала эту информацию`.

## 7. Генерация изображений

### 7.1. Модель

Рекомендуемая модель:
```text
Replicate: google/nano-banana-2
```

Альтернатива:
```text
Replicate: google/nano-banana-pro
```

### 7.2. Что генерировать

Форматы:
- обложка новости 16:9;
- квадратная картинка 1:1 для Telegram;
- изображение с крупным заголовком;
- фон в стиле Fortnite без прямого копирования официальных артов;
- стилизованный магазинный баннер;
- изображение “Новый скин скоро”;
- изображение “Утечка сезона”.

### 7.3. Безопасный визуальный стиль

Лучше не писать в промпте:
```text
copy official Fortnite art exactly
use official Fortnite logo
make it look like official Epic Games announcement
```

Лучше писать:
```text
stylized battle royale news banner, colorful game-inspired background, futuristic island, bold Russian headline, clean news design, not official branding
```

### 7.4. Промпт для генерации

```text
Create a high-quality Telegram news banner in a colorful battle royale game-inspired style.
Theme: [topic].
Mood: official gaming news, clean, energetic, premium.
Composition: central dramatic character silhouette, glowing island background, space for Russian headline.
Text on image: "[short headline in Russian]".
Do not use official Fortnite logo. Do not imitate official Epic Games branding.
Aspect ratio: 16:9.
```

```json
  "category": "skin_leak",
  "published_at": "2026-05-23T12:00:00Z",
  "is_official": false,
  "is_leak": true
}
```

### 9.3. Duplicate Checker

Задача:
- не публиковать одну и ту же новость несколько раз.

Методы:
- сравнение URL;
- сравнение заголовков;
- hash текста;
- semantic similarity через embeddings.

### 9.4. Relevance Scoring

Задача:
- определить, стоит ли новость публикации.

Ключевые слова высокого приоритета:

```text
event

leak
datamine
cosmetic
item shop
rare skin
returning skin

crew pack
V-Bucks
```

Русские эквиваленты:
```text
скин

сезон
глава

бесплатная награда
ивент
утечка
косметика
магазин предметов
редкий скин
набор
Fortnite Crew
V-Bucks
```

### 9.5. Fact/Leak Verification

Задача:
- проверять достоверность и правильно маркировать новость.

Правила:
- источник Epic/Fortnite = официально;
- Fortnite-API/Fortnite.GG = данные из базы, но не всегда официальная дата выхода;
- HYPEX/ShiinaBR/FireMonkey = утечка;
- Reddit = только дополнительное подтверждение;
- YouTube-контентер = тренд, но не источник истины.

---

### 9.6. Russian News Writer

Задача:
- писать пост на русском языке в официальном новостном стиле.

Требования:
- короткий цепляющий заголовок;
- без чрезмерного хайпа;
- без кликбейта уровня “ШОК”;
- 3–5 главных фактов;
- понятная маркировка “официально” или “утечка”;
- аккуратная ссылка на магазин.

---

### 9.7. Image Generator

Задача:
- создавать изображение для поста через Replicate.

Функции:
- `generate_news_banner(topic, headline, style)`
- `generate_shop_banner(items)`
- `generate_leak_banner(skin_name)`
- `generate_season_banner(theme)`

### 9.8. Telegram Publisher

Задача:
- публиковать посты в канал.

Функции:
- отправка текста;
- отправка изображения;
- кнопка со ссылкой на магазин;
- публикация по расписанию;
- логирование статуса.

Пример кнопки:
```text

```

## 14. Таблицы базы данных

### sources

```sql
CREATE TABLE sources (
    id SERIAL PRIMARY KEY,
    name TEXT NOT NULL,
    url TEXT NOT NULL,
    source_level INT NOT NULL,
    type TEXT NOT NULL,
    is_active BOOLEAN DEFAULT TRUE
);
```

### raw_items

```sql
CREATE TABLE raw_items (
    id SERIAL PRIMARY KEY,
    source_id INT REFERENCES sources(id),
    title TEXT,
    url TEXT UNIQUE,
    content TEXT,
    image_url TEXT,
    category TEXT,
    is_official BOOLEAN DEFAULT FALSE,
    is_leak BOOLEAN DEFAULT FALSE,
    published_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW()
);
```

### posts

```sql
CREATE TABLE posts (
    id SERIAL PRIMARY KEY,
    raw_item_id INT REFERENCES raw_items(id),
    title TEXT,
    body TEXT,
    image_prompt TEXT,
    image_url TEXT,
    score INT,
    status TEXT DEFAULT 'draft',
    created_at TIMESTAMP DEFAULT NOW()
);
```

### published_posts

```sql
CREATE TABLE published_posts (
    id SERIAL PRIMARY KEY,
    post_id INT REFERENCES posts(id),
    telegram_message_id TEXT,
    channel_id TEXT,
    published_at TIMESTAMP DEFAULT NOW()
);
```

```text
К старту сезона можно заранее пополнить V-Bucks:
```

```text
СРОЧНО КУПИ ВБАКСЫ
```

## 18. Юридические и репутационные ограничения

Важно:
- не писать, что канал официальный;
- не использовать логотип Epic Games/Fortnite так, будто это официальный канал;
- не выдавать фан-контент за официальные материалы;
- не обещать то, чего магазин не гарантирует;
- не вводить пользователя в заблуждение по поводу V-Bucks;
- не публиковать утечки как подтвержденные новости;
- для фан-контента желательно иметь дисклеймер.

Пример дисклеймера для описания канала:
```text
Неофициальный новостной канал о Fortnite. Fortnite и связанные материалы принадлежат Epic Games. Канал не связан с Epic Games.
```
